# Portfolio Website

A personal digital garden and portfolio showcasing my projects, skills, and resources. Built with a focus on performance, accessibility, and modern aesthetics using **Next.js 15**, **Tailwind CSS V4**, and **Shadcn UI**.

![Portfolio Preview](./public/preview.png)

## üöÄ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router, TurboPack)
- **Styling:** [Tailwind CSS V4](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Testing:** [Vitest](https://vitest.dev/) & [React Testing Library](https://testing-library.com/)
- **Runtime:** [Bun](https://bun.sh/)

## ‚ú® Key Features

- **Dynamic Theme:** Fully integrated Dark/Light mode with `next-themes`.
- **Responsive Design:** Mobile-first approach ensuring perfect rendering on all devices.
- **Micro-Interactions:** Subtle animations and hover effects using Framer Motion.
- **Content Sections:**
  - **Projects:** Showcase of featured work.
  - **Tech Stack:** Categorized skills (Languages vs Frameworks).
  - **Sites I Frequent:** A curated list of useful resources.
  - **Contact:** Direct links to Email, WhatsApp, GitHub, and LinkedIn.

## üõ†Ô∏è Getting Started

This project uses **Bun** for package management and script execution.

### Prerequisites

- Node.js & Bun installed.

### Installation

```bash
# Clone the repository
git clone https://github.com/manaskumar1704/Portfolio.git

# Navigate to project directory
cd Portfolio/my-app

# Install dependencies
bun install
```

### Development

Start the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Testing

Run the test suite to verify components and logic:

```bash
bun run test
```
